This project contains a web based Fedora client, with a primary focus on batch ingest.

Created by the University of Denver, Penrose Library, systems group.

As of June-2008 only a basic Fedora search and display along with a fully functional Batch Ingest is provided. The Fedora Batch Ingest is fully functional and is documented in the package
edu.du.penrose.systems.fedoraApp.batchIngest.bus

The appliations config file is ../WEB-INF/config/fedoraApp.properties
The main application log file is ../WEB-INF/logs/main.log
